Constitutive Model Behavior Using ML-FEM Hybrid Models

Project Overview:

Dec 2022 - Jun 2023

➣ A project for my MTech thesis, this project is focused on developing a Non-Linear FEM for data driven Applications using ML Algorithms.
➣ Finding out which ML algorithm is best suited for the problem and integrating it with the FEM Solver pipeline.

Role and Responsibilities:

➣ Developed a Non-Linear FEM for data driven Applications & applied on various problems (bar, beam, 2D membrane, etc.)
➣ Integrating Gaussian Process Regression, Decision Tree Regressor and Neural Network in FEM Solver pipeline
➣ Integrating Gaussian Process Regression, Decision Tree Regressor and Neural Network in FEM Solver
➣ Modified ML models to obtain predictions and their gradients, enhancing the model’s interpretability
➣ Utilized ML Models to learn constitutive behavior and implement them obtaining solutions for complex models

Results and Outcomes:

➣ Developed a Non-Linear FEM for data driven Applications using Gaussian Process Regression, Decision Tree Regressor and Neural Network
➣ Found out that Gaussian Process Regression is the best ML algorithm for the problem with small to large size data
➣ Found out that Decision Tree Regression is the best ML algorithm for problem with moderate size data
➣ Found out that Neural Network is the best ML algorithm for the problem with large to very large size data

Key Skills:

➣ Python(sklearn, pandas, numpy, matplot), Machine Learning, Data Science, Finite Element Method, Non-Linear FEM, Gaussian Process Regression, Decision Tree Regressor, Neural Network
